Installation
============

Requirements
------------

* Python_ (2.7, 3.3, 3.4, 3.5, 3.6)
* Cryptography_ (1.6+)
* Django_ (1.8, 1.10, 1.11)

.. code-block:: console

   pip install django-cryptography

.. _Cryptography: https://cryptography.io/
.. _Django: https://www.djangoproject.com/
.. _Python: https://www.python.org/
